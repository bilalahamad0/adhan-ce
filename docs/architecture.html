🌐 External services

No backend of its own, no API keys, no runtime deps. The popup calls Open-Meteo directly; the worker calls adhan-api.

🛡️ Resilience patterns

  • Dual trigger — alarm and per-tab countdown both pause; idempotent.
  • Self-resume — content.js resumes if ALARM_RESUME never fires.
  • Reconcile — worker restart re-arms an in-progress pause.
  • Stale skip — slept-through Adhan is ignored, not replayed.
  • Single instance — token kills orphaned content scripts.
  • Re-prime tabs — inject into open tabs on install & startup.
